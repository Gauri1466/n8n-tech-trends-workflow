# Daily Tech Trends Automation (n8n)

An n8n workflow that fetches daily trending tech news
from Reddit and GitHub and sends an automated email summary.

## Features
- Daily scheduled execution
- Reddit trending technology posts
- GitHub trending repositories
- Email notifications

## Tech Stack
- n8n
- Reddit API
- GitHub API
- SMTP (Email)

## Workflow Logic
Schedule Trigger → Fetch → Clean → Merge → Email
